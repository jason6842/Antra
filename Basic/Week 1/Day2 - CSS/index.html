  • Selectors

    • Element - all instances of a specific HTML element "p { color: blue; }"
    • Class - Targets elements that have a specific class assigned ".button { background-color: red; }"
    • ID - Targets a single element with a specific ID. Higher specificity than class "#header { font-size: 24px;}"
    • Descendant - Targets an element that is a descendant of another element "div p { color: green; }"
    • Group - Allows multiple selectors to be grouped together and share the same style "h1, h2, h3 { font-family: Arial, sans-serif }"
    • Attribute - Targets element with specific attributes "input[type='text'] { border: 1px solid black; }"

  • responsive Design

    • Media Queries: set break points for different devices
    • Flexbox
      • Requires a class="container"
      • display: flex; - creates two axes, main axis and cross axis
        • Items are no longer stacked on top of one another and positions along the main axis (horizontal)
      • justify-content
        • justify-content: flex-start; - aligns items along the main axis
        • justify-content: flex-end;
        • justify-content: flex-center
        • justify-content: space-between - evenly spaces items between the first and the last item
        • justify-content: space-around - similar to space-between but also distributes space for the first and last item
        • justify-content: space-evenly - evenly distributes space for all elements
      • align-items - aligns items along the cross axis (vertical)
        • align-items: flex-start; - aligns items along the cross axis
        • align-items: flex-end;
        • align-items: flex-center
      • flex-wrap: wrap; - allows items to wrap on to the next line when there is no more space available
        • flex-wrap: nowrap; - default value, items will not wrap
        • flex-wrap: wrap;
        • align-content - aligns the flex lines with the flex container, when we have flex-wrap: wrap;
          • align-content: flex-start; - aligns items along the cross axis
          • align-content: flex-end;
          • align-content: flex-center
          • align-content: space-between - evenly spaces items between the first and the last item
          • align-content: space-around - similar to space-between but also distributes space for the first and last item
          • gap: 10px; - creates space between items
          • flex-grow: 1; - allows item by element to grow and take up available space
          • flex-shrink: 1; - allows item by element to shrink and take up available space
            flex-shrink: 0; - does not allow item to shrink
          • flex-basis: 100px; - sets the initial size of the element
          • flex: 1 1 100px; - shorthand for flex-grow, flex-shrink, flex-basis
          • align-self: flex-start; - allows individual items to be aligned differently than the rest of the items
            align-self: flex-center; - allows individual items to be aligned differently than the rest of the items
            align-self: flex-end;
          • order: 1; - allows items to be reordered (should barely be used because it messes with the natural order of the DOM)
